# MohitKhare Elixir Utilities

Developer utilities and token counting tools for Elixir by [Mohit Khare](https://mohitkhare.me).

## Installation

Add `mohitkhare` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:mohitkhare, "~> 0.1.0"}
  ]
end
```

## Usage

```elixir
MohitKhare.version()
# => "0.1.0"

MohitKhare.base_url()
# => "https://mohitkhare.me"
```
